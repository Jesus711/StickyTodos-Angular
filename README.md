# StickyTodos - Angular Web Application
A lightweight, fast, and fully client-side Sticky Note Todo application built with Angular, TypeScript, Tailwind CSS, and Dexie.js. The app uses IndexedDB through Dexie.js to provide persistent storage without requiring a backend.

## Tech Stack
- Angular - Frontend framework for building the application UI and structure.
- TypeScript – Strongly typed language for better scalability and maintainability.
- Tailwind CSS – Utility-first CSS framework for fast and responsive styling.
- Dexie JS - Wrapper around IndexedDB for simple, reliable client-side data storage.

## Features
 - Create Sticky Note Todos
 - Create Note Lists
 - View all notes or view by specific list
 - Edit existing notes such as color, list, title or description
 - Delete Note and Note Lists
 -- Deleting a list will delete all notes in the list
 - Persistent local storage using IndexedDB
 - Responsive and modern UI with Tailwind CSS
